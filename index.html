Snap Card Game

This is a basic simulation of the classic card game 'Snap'.

Intructions/Guide:

To start the game, press the "Start Game" button.
This will only work for 2 Players.
Once the game starts, the deck of cards will be split between the Players. Then, the game will automatically reveal the top card of each Player's Hand.
The game will then wait for a couple seconds, to allow for Players to claim a 'Snap' - if there is one.
A 'Snap' is when there is a matching number or letter on the two showing cards. EXAMPLE: Player 1 has a 2 of Hearts, and Player 2 has a 2 of Clubs. This would be a Snap as both are showing the number 2.
The game ends when one Player has all the cards (and thus, wins).

Player One's Snap Call button is "a"; whereas Player Two's Call button is "l".
